# 1. Product Class
## Responsibilities:
- Store information about individual products.
- Manage operations like adding new products, updating stock levels, and retrieving product information.
# 2. Order Class
## Responsibilities:
- Track customer orders.
- Manage operations like adding new orders, updating order statuses, and retrieving order information.
# 3. Supplier Class
## Responsibilities:
- Manage supplier information.
- Track and update supplier performance metrics.
# 4. InventoryDB Class (Found in the main folder)
## Responsibilities:
- Handle database connections.
- Manage CRUD (Create, Read, Update, Delete) operations for products, orders, and suppliers.
- Use SQLite for persistent data storage.
