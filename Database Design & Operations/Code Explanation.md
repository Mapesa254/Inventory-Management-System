# Summary of the database Structure:
* database.py: Contains the Database class, which handles general database operations such as creating connections, executing queries, and fetching data.

* inventory_db.py: Contains the InventoryDB class, which extends the Database class to add specific functionality for managing products, orders, and suppliers in an inventory system.

* main.py: The entry point of the application, where the InventoryDB class is used to demonstrate adding products, updating stock levels, placing orders, and managing suppliers.
