# Database Design & Operations
- In this section, I focused on creating a robust and well-structured database using SQLite. 
- The design encapsulates all essential aspects of inventory management, including product details, stock levels, suppliers, and order histories. 
- The database schema is carefully designed to ensure data integrity and support efficient querying. 
- Additionally, I implemented various functions in a modular way, using Object-Oriented Programming (OOP) principles to facilitate database operations like adding products, updating stock, and retrieving inventory data.
